picPySort

Goal of this tool is to sort/organize photos on the disk in folders named after date or place where the photo was taken, if EXIF of a file contains such information.


What you need to run it?
	Of course python installed, development was started on Pyhton 3.6
And some additional libraries (their names, at the end of the commands how to add them):
	pip install exifread
	pip install requests
	pip install geopy
